What is Insertion Sort?

Insertion Sort is a simple sorting algorithm that builds a sorted array one element at a time. It is much less efficient on large lists than more advanced algorithms such as quicksort, heapsort, or merge sort.

How Insertion Sort Works

The algorithm divides the input list into two parts: a sorted part and an unsorted part. It iteratively takes an element from the unsorted part and inserts it into the correct position in the sorted part.

Insertion Sort Algorithm Steps

  1. Start from the second element (index 1) of the array.
  2. Compare the current element with the elements in the sorted part (to its left).
  3. Shift all larger elements to the right to make space for the current element.
  4. Insert the current element into its correct position.
  5. Repeat until the entire array is sorted.

Insertion Sort Implementation


function insertionSort(arr) {
    for (let i = 1; i < arr.length; i++) {
        let key = arr[i];
        let j = i - 1;

        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j--;
        }
        arr[j + 1] = key;
    }
    return arr;
}

Time Complexity

The average and worst-case time complexity of Insertion Sort is O(n²), where n is the number of items being sorted. The best-case time complexity is O(n), which occurs when the array is already sorted.
