Overview

Quicksort is a highly efficient sorting algorithm and is based on partitioning an array into smaller sub-arrays. A sub-array is sorted recursively, and the base case is when the size of the sub-array is one or zero.

How Quicksort Works

The algorithm follows these steps:

  1. Choose a pivot element from the array.
  2. Partition the array into two halves: elements less than the pivot and elements greater than the pivot.
  3. Recursively apply the above steps to the sub-arrays.

Implementation


function quicksort(arr) {
    if (arr.length <= 1) {
        return arr;
    }
    const pivot = arr[arr.length - 1];
    const left = [];
    const right = [];
    for (let i = 0; i < arr.length - 1; i++) {
        if (arr[i] < pivot) {
            left.push(arr[i]);
        } else {
            right.push(arr[i]);
        }
    }
    return [...quicksort(left), pivot, ...quicksort(right)];
}

Time Complexity

The average-case time complexity of Quicksort is O(n log n), while the worst-case time complexity is O(n²), which occurs when the smallest or largest element is always chosen as the pivot.
