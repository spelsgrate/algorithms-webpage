What is Linear Search?

Linear search is a simple search algorithm that checks every element in a list until the desired element is found or the list ends.

How Linear Search Works

The algorithm starts at the beginning of the list and compares each element with the target value. If a match is found, the index of the element is returned. If the end of the list is reached without finding the target, a value indicating failure (often -1) is returned.

Implementation


function linearSearch(arr, target) {
    for (let i = 0; i < arr.length; i++) {
        if (arr[i] === target) {
            return i; // Target found
        }
    }
    return -1; // Target not found
}

Complexity Analysis

The time complexity of linear search is O(n), where n is the number of elements in the list. The space complexity is O(1) since no additional storage is required.
