Overview

Radix Sort is a non-comparative integer sorting algorithm that sorts numbers by processing individual digits. It works by grouping numbers by their individual digits, starting from the least significant digit to the most significant digit.

How Radix Sort Works

The algorithm processes each digit of the numbers in a specific base (usually base 10). It uses a stable sorting algorithm (like Counting Sort) as a subroutine to sort the numbers based on each digit.

  1. Find the maximum number to determine the number of digits.
  2. For each digit (from least significant to most significant), perform a stable sort on the array.
  3. Repeat until all digits are processed.

Implementation

function countingSortForRadix(arr, exp) {
    let output = new Array(arr.length);
    let count = new Array(10).fill(0);

    for (let i = 0; i < arr.length; i++) {
        count[Math.floor(arr[i] / exp) % 10]++;
    }

    for (let i = 1; i < 10; i++) {
        count[i] += count[i - 1];
    }

    for (let i = arr.length - 1; i >= 0; i--) {
        output[count[Math.floor(arr[i] / exp) % 10] - 1] = arr[i];
        count[Math.floor(arr[i] / exp) % 10]--;
    }

    for (let i = 0; i < arr.length; i++) {
        arr[i] = output[i];
    }
}

function radixSort(arr) {
    let max = Math.max(...arr);

    for (let exp = 1; Math.floor(max / exp) > 0; exp *= 10) {
        countingSortForRadix(arr, exp);
    }
}

Complexity Analysis

The time complexity of Radix Sort is O(d * (n + k)), where:

  • d is the number of digits in the maximum number.
  • n is the number of elements in the array.
  • k is the range of the input.

The space complexity is O(n + k).
