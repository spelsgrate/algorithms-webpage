Overview

The Floyd-Warshall algorithm is a dynamic programming algorithm used to find the shortest paths in a weighted graph with positive or negative edge weights (but with no negative cycles). It can find the shortest paths between all pairs of vertices in a graph.

Algorithm

The algorithm works by considering all pairs of vertices and systematically updating the shortest path between them. It uses a 2D array to store the shortest path distances and iteratively improves these distances.

Implementation

function floydWarshall(graph) {
    const dist = [];
    const V = graph.length;

    for (let i = 0; i < V; i++) {
        dist[i] = [];
        for (let j = 0; j < V; j++) {
            dist[i][j] = graph[i][j];
        }
    }

    for (let k = 0; k < V; k++) {
        for (let i = 0; i < V; i++) {
            for (let j = 0; j < V; j++) {
                if (dist[i][k] + dist[k][j] < dist[i][j]) {
                    dist[i][j] = dist[i][k] + dist[k][j];
                }
            }
        }
    }
    return dist;
}

Complexity

The time complexity of the Floyd-Warshall algorithm is O(V^3), where V is the number of vertices in the graph. The space complexity is O(V^2) due to the storage of the distance matrix.
